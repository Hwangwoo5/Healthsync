
# Hwangyeong_i - 아두이노 환경 센서 제어 모듈

이 디렉토리는 아두이노를 이용한 환경 센서 연동 코드 `Hwangyung_ii.ino`를 포함하고 있습니다.  
해당 코드는 온습도, 가스, 먼지 등 환경 요소를 측정하고 시리얼 통신으로 출력하는 기능을 수행합니다.

---

## 📦 사용 센서 목록 (예시)

- DHT11 / DHT22 (온습도 센서)
- MQ 시리즈 (가스 센서)
- GP2Y1010AU0F (먼지 센서)
- 조도 센서 (포토레지스터)

※ 실제 연결된 센서는 `Hwangyung_ii.ino` 코드 내 핀 설정과 주석을 참고하세요.

---

## ⚡ 주요 기능

- 아날로그/디지털 센서 데이터를 주기적으로 읽어 시리얼로 출력
- 환경 이상 수치 감지 시 알림 출력 또는 외부 장치와 연동 가능
- 추후 Raspberry Pi / Jetson과 USB 시리얼 통신 가능

---

## 🔌 하드웨어 구성도 (예시)

- 5V → 센서 VCC
- GND → 센서 GND
- A0 / D2 등 → 각 센서의 데이터 핀

```plaintext
[ Arduino UNO ]
      |
      |-- D2 : PIR Sensor
      |-- A0 : Gas Sensor
      |-- A1 : Light Sensor
      |-- D3 : Buzzer (optional)
```
